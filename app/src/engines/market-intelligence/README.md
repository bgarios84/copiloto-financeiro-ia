# src/engines/market-intelligence

Motor de inteligência de mercado.

Responsabilidades:
- Coleta e processamento de dados de mercado em tempo real
- Análise de indicadores macroeconômicos (Selic, IPCA, CDI, etc.)
- Monitoramento de notícias e eventos relevantes
- Análise de sentimento de mercado
- Alertas de movimentos significativos em ativos monitorados

Exemplos de funções:
- `fetchMarketData(assets)`
- `getEconomicIndicators()`
- `analyzeSentiment(newsItems)`
- `monitorWatchlist(userId)`
