# src/engines/finance

Motor de cálculos financeiros fundamentais.

Responsabilidades:
- Cálculo de juros simples e compostos
- Projeções de fluxo de caixa
- Análise de orçamento vs. realizado
- Cálculo de índices financeiros (liquidez, endividamento, etc.)
- Categorização e classificação de transações

Exemplos de funções:
- `calculateCompoundInterest(principal, rate, period)`
- `projectCashFlow(transactions, horizon)`
- `analyzeBudgetVariance(budget, actual)`
