# src/engines/investments

Motor de análise de investimentos.

Responsabilidades:
- Cálculo de rentabilidade (absoluta e relativa a benchmarks)
- Análise de risco (volatilidade, VaR, Sharpe ratio)
- Comparação entre produtos de investimento
- Simulação de aportes e resgates
- Análise de renda fixa (LCI, LCA, CDB, Tesouro Direto)
- Análise de renda variável (ações, FIIs)

Exemplos de funções:
- `calculateReturn(portfolio, period)`
- `calculateVolatility(prices)`
- `compareWithBenchmark(portfolio, benchmark)`
