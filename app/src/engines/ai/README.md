# src/engines/ai

Motor de Inteligência Artificial financeira.

Responsabilidades:
- Análise e categorização automática de transações via IA
- Geração de insights financeiros personalizados
- Detecção de padrões de gastos e anomalias
- Respostas do assistente financeiro (chat)
- Previsão de gastos e receitas futuros

Exemplos de funções:
- `categorizeTransaction(transaction)`
- `generateInsights(financialData)`
- `detectAnomalies(transactions)`
- `forecastExpenses(history, horizon)`
