# src/engines/portfolio

Motor de gestão e análise de portfólio.

Responsabilidades:
- Consolidação de ativos de múltiplas contas e corretoras
- Análise de alocação e diversificação
- Rebalanceamento de carteira
- Cálculo de patrimônio líquido consolidado
- Análise de concentração de risco por ativo/setor/classe

Exemplos de funções:
- `consolidatePortfolio(accounts)`
- `analyzeAllocation(portfolio)`
- `suggestRebalancing(portfolio, targetAllocation)`
